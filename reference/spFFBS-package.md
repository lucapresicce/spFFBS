# spFFBS: Spatiotemporal Propagation for Multivariate Bayesian Dynamic Learning

Provides parallelized implementation of forward filtering backward
sampling algorithm, along with dynamic Bayesian predictive stacking to
achieve exact posterior inference avoiding simulation-based approaches.
