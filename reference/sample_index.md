# Function to sample integers (index)

Function to sample integers (index)

## Usage

``` r
sample_index(size, length, p)
```

## Arguments

- size:

  [integer](https://rdrr.io/r/base/integer.html) dimension of the set to
  sample

- length:

  [integer](https://rdrr.io/r/base/integer.html) number of elements to
  sample

- p:

  [vector](https://rdrr.io/r/base/vector.html) sampling probabilities

## Value

[vector](https://rdrr.io/r/base/vector.html) sample of integers
